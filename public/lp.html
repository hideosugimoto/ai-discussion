よくある質問

本当に Claude / ChatGPT / Gemini の最新モデルが使えますか?

はい。Claude Opus 4.6 / GPT-4o / Gemini 2.5 Pro など、各社の最新モデルを使用しています。高速モードでは sonnet / mini / flash に切替可能です。

API キーは必要ですか?

Premium プラン (¥980/月) なら不要です。Google ログインだけで即座に利用開始できます。Free プランの場合のみ、各社の API キーをご自身で取得してブラウザに保存する形になります。

月の使用量に上限はありますか?

あります。Premium は月 $5 相当、Plus は月 $10 相当のトークン上限を設定しています。超過しそうな場合は追加クレジット (¥500 = +$2) を購入できます。一般的な使い方なら Premium の上限内で十分です。

議論データはどこに保存されますか?

Free プランは IndexedDB（ブラウザ内）のみ。Premium プランでは Cloudflare D1 にクラウド同期されますが、暗号化通信で保護され、共有リンクで公開する際にもプロフィール・憲法・APIキーなどの個人情報は自動的に除外されます。

解約はいつでもできますか?

はい。Stripe のカスタマーポータルからワンクリックで解約できます。日割り計算はありませんが、当月末まで利用可能です。

ソースコードは公開されていますか?

はい。GitHub で全て公開しています（BSL 1.1 ライセンス）。個人利用・学習・改変は自由です。2030年4月以降は MIT License に自動移行します。商用利用には商用ライセンスが必要です。

セルフホストできますか?

はい。Cloudflare Pages にデプロイすれば自分専用のインスタンスを立てられます。README にセットアップ手順を記載しています。
